# 데이터 센터 유형
## 데이터 센터란?
데이터 센터는 서버, 네트워크, 스토리지 등 IT 서비스 제공에 필요한 장비를 통합 관리하는 시설을 말합니다. 아래는 [위키피디아](https://ko.wikipedia.org/wiki/%EB%8D%B0%EC%9D%B4%ED%84%B0_%EC%84%BC%ED%84%B0)의 설명입니다.

> 데이터 센터(data center)는 서버 컴퓨터와 네트워크 회선 등을 제공하는 건물이나 시설을 말한다. 서버 호텔(server hotel)이라고도 부른다.
데이터 센터는 인터넷의 보급과 함께 폭발적으로 성장하기 시작했다. 인터넷 검색, 쇼핑, 게임, 교육 등 방대한 정보를 저장하고 웹 사이트에 표시하기 위해
수천, 수만 대의 서버 컴퓨터가 필요하게 되자, 이 서버 컴퓨터를 한 장소에 모아 안정적으로 관리하기 위한 목적으로 인터넷 데이터 센터를 건립하게 되었다.
통신업체의 데이터 센터는 인터넷 데이터 센터(Internet data center, IDC), 클라우드 서비스를 위한 데이터 센터는 클라우드 데이터 센터(cloud data center)로 부르기도 하나,
요즘 이러한 용어들의 구분은 거의 사라지고 데이터 센터라는 용어 하나로 통합되어 쓰이고 있다.

세계 인터넷 트래픽 증가 추세에 맞춰 세계의 데이터 센터 수요도 최근 몇년 사이에 급증하고 있습니다. 현재는 전 세계에 8,200개의 데이터 센터가 있습니다. 그 중 3분의 1이 미국, 버지니아 주에 분포해있습니다. 전세계 인터넷 트래픽의 70%가 버지니아 주에서 발생하기 때문입니다.
미국에서는 [에퀴닉스(Equinix)](https://www.equinix.kr/), [디지털 리얼티(Digital Realty)](https://www.digitalrealty.com/) 등 데이터 센터 리츠의 수익률이 과거에 비해 높아졌습니다.
한국은 전력의 품질의 높은 반면 전기료는 높지 않은 편이기 때문에, 국내 데이터 센터도 증가 추세입니다.

## 데이터 센터의 유형
데이터 센터는 사용 목적에 따라서 크게 두 가지로 분류됩니다. 하나는 기업, 정부, 금융권의 사용을 목적으로 하는 자사형 데이터 센터(Enterprise Data Center)입니다. 또 하나는 수익 창출이 목적인 인터넷 데이터 센터(Internet Data Center, IDC)입니다. KT, LG, U+와 같은 통신사업자의 데이터 센터, 네이버나 카카오와 같은 IT 기업의 데이터 센터가 인터넷 데이터 센터에 해당됩니다.

### 자사용 데이터 센터
자사용 데이터 센터는 정보의 안전한 보관과 안정성이 최우선입니다. 자사용 데이터의 대표적인 사례가 금융권의 데이터 센터입니다. 금융권 데이터 센터는 아래와 같이 3 개의 데이터 센터를 구축해 운영합니다.

* 메인 센터(Main ceter): 평상시 운영
* 재난 복구 센터(Disaster Recovery center): 비상시 운영
* 백업 센터(Backup center): 데이터 백업

금융권의 데이터 센터는 메인 센터를 평상시에 운영합니다. 재난 복구 센터는 실시간으로 메인센터를 동일하게 미러링합니다. 그래서 메인 센터가 정상적으로 기능하지 않을 때에는 50Km 이내에 위치한 재난 복구 센터가 서비스를 제공하는 구조입니다. 전쟁이나 자연재해와 같은 경우에는 백업 센터에 저장된 데이터를 불러오게 됩니다.
### 인터넷 데이터 센터
인터넷 데이터 센터는 수많은 정보가 이동하는 것으로 이익을 내기 때문에, 수익성과 효율성 두 가지가 중요합니다.

## 데이터 센터 규모에 따른 분류
데이터 센터를 규모에 따라 분류하기도 합니다. 규모로 분류하면 소형, 중형, 대형, 거대(massive), 메가(mega)로 분류합니다. 고직접, 고민도 서버와 더불어 IDC가 갈수록 대형화되는 추세입니다. 최근에는 자체 변전소가 있는 하이퍼 스케일(hyper scale) 데이터 센터도 많이 구축되고 있습니다.
* 소형
* 중형
* 대형
* 거대
* 메가
* 하이퍼 스케일

### 전력부하밀도
전력부하밀도란, 1개의 랙에 설치된 서버들이 사용하는 전력량(kW/rack)을 말합니다. 과거의 데이터 센터는 3kw 내외의 저밀도 랙으로 계획된 반면에, 최근에는 고밀도, 고집적 서버가 계속 개발되면서 8kw 이상 혹은 20kw 이상을 사용할 수 있도록 고밀도 데이터 센터로 계획해서 효율성을 높입니다.
