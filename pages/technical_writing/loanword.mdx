# 이상하고 어색한 외래어 표기
IT와 관련된 용어들은 대부분 영어로 되어있습니다. 그래서 테크니컬 라이팅을 하다 보면 외래어 표기를 해야 할 경우가 많습니다.
그럴 때마다 한 번쯤은 고민하게 됩니다. 왜냐하면 국립국어원에서 규정한 외래어 표기가 현실에서 사람들이 사용하는 표기와 다른 경우가 많기 때문입니다.

![피카츄의 올바른 표기는 피카추](/images/technical_writing/technical_writing_01.jpeg)
<sub>피카츄의 올바른 표기는 피카추, MBC 우리말 나들이</sub>

이 글에서는 IT와 관련된 용어의 외래어 표기를 정리합니다.
아울러 국립국어원의 외래어 표기가 현실에 정착한 표기와, 국립국어원의 외래어 표기가 아직은 어색하고 낯설게 느껴지는 표기에 대해서 알아봅니다.

<Callout type="info" emoji="💡">
   이 글은 수시로 업데이트됩니다.
</Callout>

## 익숙한 외래어 표기
아래는 국립국어원의 표기가 거의 정착한 경우에 해당하는 경우입니다.
### A~D
| 영문          | 틀린 표현  | 맞는 표현  |
|-------------|--------|--------|
| appliaction | 어플리케이션 | 애플리케이션 |
| contents    | 컨텐츠    | 콘텐츠    |
| component   | 콤포넌트   | 컴포넌트   |
| cabinet     | 캐비넷    | 캐비닛    |
| collection  | 콜렉션    | 컬렉션    |

### E~Z
| 영문          | 틀린 표현  | 맞는 표현  |
|-------------|--------|--------|
| front       | 프론트    | 프런트    |
| message     | 메세지    | 메시지    |
| navigator   | 네비게이터  | 내비게이터  |
| repository  | 리파지토리, 레포지토리 | 리포지토리 |


## 여전히 어색한 외래어 표기
아래는 사람들이 국립국어원의 표기를 어색하게 느끼고 잘 쓰지 않는 경우입니다.

| 영문        | 틀린 표현    | 맞는 표현 |
|-----------|----------|-------|
| solution  | 솔루션      | 설루션   |
| thumbnail | 썸네일      | 섬네일   |
| parallel  | 페러렐, 패러랠 | 패럴렐   |
| proxy     | 프록시      | 프락시   |
